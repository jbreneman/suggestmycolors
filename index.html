Suggestmycolors was written a long time ago back when I was really into colourlovers.com. I created this app that would apply math to a color that you picked and throw a bunch of colors back at you. At the time, I didn't know much javascript so it was written in php. Yeah. After a while I picked up a little jQuery and was able to hook up my php script with a little ajax magic to get it to be a bit more responsive. This is v3, and it's written in pure javascript. I'm in the process of learning javascript and I decided to redo this little project as a way of diving in headfirst. So far it's worked, I know a lot more about javascript than I did!

Some info on the different labels: The opposites are just that, opposites of the red, green, and blue values. Half opposites are the red, green, and blue values with a half of the color scale added to them. Red, green, and blue shifts are shifts toward those colors. Check out #000000 to get an idea of how this works.

This experiment should work in all modern browsers (Chrome, Safari, Firefox, Opera, IE 9+).
